🌿 Plant Disease Classifier
📚 Overview

Plant Disease Classifier is a deep learning–based web application that identifies plant leaf diseases from images using a Convolutional Neural Network (CNN). The application leverages regularization techniques to improve generalization and reduce overfitting, making it reliable for real-world usage.

🔧 Features

The project follows an end-to-end deep learning pipeline, beginning with image preprocessing and data augmentation to increase dataset diversity. A CNN model is trained with Dropout layers to minimize overfitting and enhance model robustness. The trained model is deployed through an interactive Streamlit web interface, allowing users to upload leaf images and receive disease predictions in real time.

🛠️ Technologies Used

This project utilizes Python, TensorFlow/Keras, NumPy, Pandas, and OpenCV for image processing and deep learning. Matplotlib and Seaborn are used for visualization and performance analysis. Model development and experimentation are conducted in Jupyter Notebook, while Streamlit and Render are used for deployment and public access.
